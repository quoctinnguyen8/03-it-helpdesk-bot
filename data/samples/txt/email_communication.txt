==================== OUTLOOK ISSUES ====================

English:
Common Microsoft Outlook problems and solutions:

1. Outlook won't open / keeps crashing:
   - Start Outlook in Safe Mode: Hold Ctrl while opening, or run "outlook.exe /safe"
   - Disable add-ins: File > Options > Add-ins > Manage COM Add-ins
   - Create new Outlook profile: Control Panel > Mail > Show Profiles > Add
   - Repair Office installation
   - Run Outlook diagnostics: support.microsoft.com/scanpst

2. Cannot send or receive emails:
   - Check internet connection
   - Verify account settings: File > Account Settings
   - Check if mailbox is full (delete old emails)
   - Disable antivirus email scanning temporarily
   - Test with Outlook safe mode
   - Check server settings with email provider

3. Outlook is slow or freezing:
   - Compact mailbox: Right-click folder > Properties > Advanced > Compact Now
   - Archive old emails: File > Cleanup Tools > Archive
   - Reduce mailbox size (delete large attachments)
   - Disable RSS feeds and shared calendars
   - Turn off cached mode temporarily
   - Repair PST/OST file with SCANPST.EXE

4. Emails stuck in Outbox:
   - Go to Outbox folder
   - Delete problematic email (especially large attachments)
   - Restart Outlook
   - Check if you're working offline (Send/Receive tab)

5. Search not working:
   - Rebuild search index: Control Panel > Indexing Options > Advanced > Rebuild
   - Check if Outlook is in index: Indexing Options > Modify
   - Repair Office

6. AutoComplete not working:
   - Delete NK2 file or AutoComplete cache
   - File > Options > Mail > Empty Auto-Complete List
   - Reset autocomplete suggestions

==================== GMAIL ISSUES ====================

English:
Fixing Gmail problems:

1. Cannot access Gmail:
   - Check if Gmail is down: downdetector.com
   - Clear browser cache and cookies
   - Try incognito/private mode
   - Try different browser
   - Check internet connection
   - Disable browser extensions

2. Not receiving emails:
   - Check Spam folder
   - Check filters: Settings > Filters and Blocked Addresses
   - Check forwarding settings
   - Check storage quota (15GB free limit)
   - Delete old emails or upgrade storage
   - Check if sender is not blocked

3. Cannot send emails:
   - Check attachment size (25MB limit)
   - Use Google Drive for large files
   - Check if recipient address is correct
   - Check daily sending limit (500 emails/day for free accounts)
   - Verify account (add phone number)

4. Gmail search not working:
   - Use advanced search operators
   - Check spelling
   - Search in All Mail instead of specific folder
   - Clear browser cache

5. Slow performance:
   - Disable chat
   - Reduce inbox density: Settings > Display density > Compact
   - Use basic HTML version: gmail.com/mail/u/0/h
   - Clear browser cache
   - Disable labs features

6. Account security issues:
   - Enable 2-Step Verification
   - Review recent security activity
   - Check connected devices and apps
   - Update recovery email and phone
   - Use app-specific passwords for third-party apps

   - Xóa browser cache
   - Tắt labs features

6. Vấn đề bảo mật tài khoản:
   - Bật 2-Step Verification
   - Xem lại hoạt động bảo mật gần đây
   - Kiểm tra thiết bị và apps kết nối
   - Cập nhật email và phone khôi phục
   - Dùng app-specific passwords

==================== ZOOM / VIDEO CONFERENCING ====================

English:
Troubleshooting video conferencing issues:

Audio problems:
1. Cannot hear others:
   - Check if muted in Zoom
   - Check system volume
   - Select correct audio device: Settings > Audio > Speaker
   - Test speaker: Settings > Audio > Test Speaker
   - Update audio drivers

2. Others cannot hear you:
   - Check if microphone is muted
   - Select correct microphone: Settings > Audio > Microphone
   - Test microphone: Settings > Audio > Test Mic
   - Check microphone permissions in Windows
   - Try different USB port for external mic
   - Update audio drivers

3. Echo or feedback:
   - Use headphones
   - Mute when not speaking
   - Reduce speaker volume
   - Move microphone away from speakers

Video problems:
1. Camera not working:
   - Check if camera is enabled in Zoom
   - Select correct camera: Settings > Video > Camera
   - Check camera privacy settings in Windows
   - Close other apps using camera
   - Update camera drivers
   - Restart computer

2. Poor video quality:
   - Check internet speed (need 3+ Mbps for HD video)
   - Enable HD video: Settings > Video > Enable HD
   - Turn off other participants' videos
   - Close bandwidth-heavy applications
   - Use wired connection instead of WiFi

Connection problems:
1. Meeting disconnects frequently:
   - Check internet stability
   - Use wired connection
   - Close unnecessary applications
   - Update Zoom to latest version
   - Try different network

2. Slow or lagging:
   - Turn off video
   - Lower video quality
   - Close other applications
   - Check for network congestion
   - Restart router

Screen sharing issues:
1. Cannot share screen:
   - Update Zoom
   - Check screen sharing permissions
   - Try different sharing option (desktop vs. window)
   - Restart Zoom

2. Choppy screen sharing:
   - Optimize for full screen video clip
   - Reduce video quality
   - Close unnecessary applications
   - Check CPU usage
